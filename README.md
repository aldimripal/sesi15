#  Praktikum Algoritma dan Struktur Data

## Binary Search

Repository ini berisi hasil praktikum mata kuliah **Algoritma dan Struktur Data** dengan materi **Binary Search**, meliputi implementasi pada data numerik dan string, visualisasi langkah pencarian, serta studi kasus pencarian nilai mahasiswa.


##  Aktivitas 1 – Menyiapkan Data Terurut

Pada aktivitas ini dilakukan proses **pengurutan data** sebagai tahap awal sebelum menerapkan algoritma Binary Search. Binary Search hanya dapat bekerja pada data yang telah terurut.

Data berupa array bilangan bulat diurutkan secara **ascending** menggunakan metode sorting. Hasil pengurutan ini menjadi dasar untuk aktivitas-aktivitas selanjutnya.

**Tujuan aktivitas:**

* Memahami pentingnya data terurut
* Menyiapkan data yang valid untuk Binary Search


##  Aktivitas 2 – Binary Search Dasar (Data Angka)

Aktivitas ini berfokus pada implementasi **algoritma Binary Search** untuk mencari sebuah nilai pada array bertipe integer.

Program dibuat dalam bahasa **Java**, di mana sistem akan:

* Mencari nilai tertentu
* Menampilkan indeks hasil pencarian
* Menentukan status apakah data ditemukan atau tidak

**Tujuan aktivitas:**

* Memahami logika dasar Binary Search
* Mengimplementasikan pencarian data numerik secara efisien

---

##  Aktivitas 3 – Visualisasi Langkah Binary Search

Pada aktivitas ini, algoritma Binary Search dikembangkan dengan menampilkan nilai **low, mid, dan high** di setiap iterasi perulangan.

Visualisasi ini bertujuan untuk memperlihatkan bagaimana ruang pencarian terus dipersempit hingga data ditemukan atau tidak ditemukan.

**Tujuan aktivitas:**

* Memahami alur kerja Binary Search secara detail
* Menganalisis jumlah langkah pencarian
* Membandingkan efisiensi dengan Linear Search

---

##  Aktivitas 4 – Binary Search pada Data String

Aktivitas ini menerapkan Binary Search pada data bertipe **String**. Proses pencarian menggunakan metode `compareToIgnoreCase()` agar pencarian tidak sensitif terhadap perbedaan huruf besar dan kecil.

Dengan demikian, pencarian menjadi lebih fleksibel dan akurat.

**Tujuan aktivitas:**

* Mengimplementasikan Binary Search pada data non-numerik
* Memahami penggunaan metode perbandingan string
* Menangani perbedaan kapitalisasi teks

---

##  Aktivitas 5 – Mini Challenge: Sistem Pencarian Nilai Mahasiswa

Pada mini challenge ini dibuat sebuah sistem sederhana untuk mencari **nilai mahasiswa** menggunakan Binary Search.

Program akan:

* Menyimpan array nilai yang sudah terurut
* Mencari nilai tertentu
* Menampilkan indeks nilai
* Menghitung jumlah langkah pencarian
* Menampilkan status ditemukan atau tidak ditemukan

Hasil pengujian program dan source code diunggah ke repository GitHub sebagai bagian dari dokumentasi praktikum.
